## API LmStore
Uma API criada para um Ecommerce de games, no qual terá os melhores jogos da
atualidade.
 <br/> <br/>
 O projeto conta com diversas funcionalidades, sendo algumas como: opção de compra, através de cartão de crédito ou débito, o envio de e-mail para confirmação de conta, para confirmação de compra e também para recuperação de senha. Após efetuar uma compra, o usuário também poderá deixar um feedback sobre o jogo.
 <br/> 
 A API também conta com o admin, no qual apenas ele poderá ter acesso a todos os pedidos, adicionar novos produtos e etc...
 <br/> <br/>
Esta API está será consumida por um projeto que ainda está em desenvolvimento. Algumas partes do código foram removidas, pois em breve postarei um video mostrando como foi feito o projeto.
